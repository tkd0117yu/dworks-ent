[内容]		データ書き出しとメモリ確保

*************************************************************

カンマ区切りのデータ(TestData.txt)を読み込み、
バイナリデータとして書き出す。
データ形式は以下の通り。

読み込むデータ列の数、読み込むファイル名、書き出すファイル名前を
実行ファイルへ渡す引数から取得すること。

読み込むデータ列の数分のメモリを確保し、
すべてが完了したらメモリを返却すること。


------------------------------------------------------

{
	数値データ      : 4Byte
	数値データ		: 4Byte
}
