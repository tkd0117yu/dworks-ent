			データ結合と復元

// 課題6
ファイルリスト(FileList.txt)を読み込み、
そこに記述されたファイルをバイナリ結合するツール（①）を作成する。

ファイルリストは実行ファイルへの引数から取得し、
第二引数として指定したファイル名前で結合して出力すること。


// 課題7
また、その結合したファイルを元のデータに復元するツール（②）を作成。
②は①で結合されたバイナリデータならば、
どんなものでも復元できるようにすること。

余裕があるならば、結合されたファイルから
結合番号、もしくはファイル名を指定し、
そのファイルのみを抜き出せるようにする。
