[内容]		バイナリデータ基礎

*************************************************************

バイナリデータ(TestData.dat)を読み込み、コンソールにその内容を出力する。
バイナリデータの形式は以下の通り。
読み込むファイルは実行ファイルへ渡す引数から取得すること。

------------------------------------------------------

データ総数(DataNum) : 4Byte

DataNum分以下のデータが続く
{
	文字列の長さ(n) : 2Byte
	文字列          : nByte
	数値データ      : 4Byte
}

------------------------------------------------------
